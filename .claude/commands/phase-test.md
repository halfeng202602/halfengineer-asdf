現在のテストを全実行し、結果サマリーをtasks.mdに追記する。

## 手順

1. プロジェクトのテストフレームワークを確認する（CLAUDE.md の Commands セクション参照）
2. テストを全実行する
3. 結果を以下のフォーマットでサマリーする：

```
テスト実行結果:
  実行日時: {YYYY-MM-DD HH:mm}
  総テスト数: {N}
  パス: {N}
  フェイル: {N}
  スキップ: {N}
  カバレッジ: {N}%（取得可能な場合）
```

4. 失敗テストがある場合：
   - 失敗したテスト名と失敗理由を列挙する
   - 原因の推定と修正案を提示する

5. 結果サマリーを specs/{feature}/tasks.md の該当タスクに追記する

## 注意
- テストの修正は行わない。結果の報告のみ。
- 修正が必要な場合はPOに報告する。
