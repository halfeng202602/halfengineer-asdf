tasks.md を読み取り、現在の進捗をサマリー表示する。

## 手順

1. specs/ 配下の全 feature ディレクトリを探索する
2. 各 feature の tasks.md を読み取る
3. 以下をサマリー表示する：

```
📊 プロジェクトステータス

## 現在のフェーズ
{Phase名}（{feature名}）

## タスク進捗
完了: {N} / 全体: {N}
  ✅ DONE: {タスク一覧}
  🔄 IN_PROGRESS: {タスク一覧}
  ⏳ TODO: {タスク一覧}

## バグ記録
bugs/ 内のファイル数: {N}
  - {バグファイル一覧}

## 最後のレビュー結果
{APPROVE / REQUEST_CHANGES / 未実施}
{review.md のパス}

## 注意事項
{未解決の問題があれば}
```

4. 情報がない場合は「未開始」と表示する

## 注意
- 読み取りのみ。ファイルの変更は行わない。
